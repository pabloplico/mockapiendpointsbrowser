How this works
  1. Paste your Gemini API key below. It's saved to localStorage and only sent to Google.
  2. Click + New endpoint, set the method + path, and describe the data you want.
  3. Click Generate. Review the JSON. Approve it to go live, or Generate again to retry.
  4. Any page on this same origin that calls fetch('/your/path') will receive the approved mock — your app needs no changes.
Statuses: draft no data yet · generated review it · approved live.
Editing method, path, or description un-approves the mock so you can't accidentally serve stale data.
